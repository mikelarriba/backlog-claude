## IR31591525 AM BI – MIDAS – Discrepancy of file sizes between direct download of test files and data export via Isilon

### Description

The user is reporting the following problem:

 

The user reports a discrepancy in file sizes when using the new direct download function from the test detail view. During testing, the user observed that the file sizes obtained via direct download differ from those resulting from a data export using the Isilon link for the same test.

 

After downloading all files directly from the test detail view into the download area and extracting them, the resulting file sizes did not match those from the Isilon export. Screenshots illustrating the discrepancy have been provided by the user for reference.

 

The user requests to check this issue, and if necessary hand the ticket over to the MIDAS-developers.
